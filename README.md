## 🛠️ Stack Technique
- **SQL (MySQL)** : Création des tables `label_stats` et `studio_business`.
- **Power BI** : Analyse du Chiffre d'Affaires (**420 000 FCFA**) et des vues YouTube.
- **Git/GitHub** : Gestion de projet et documentation.
